Weekly GOAT Competition

üí∞ How Payouts Work

Adding New Chores

To add a permanent chore (requires database access):

  1. Go to your Supabase dashboard
  2. Navigate to the chores table
  3. Click "Insert row"
  4. Fill in: name, base_value (points), max_per_day (usually 1), icon (emoji), cooldown_hours
  5. Save and refresh the app

For one-time tasks, use the "Custom Task" feature in Admin mode!

‚ùì Frequently Asked Questions

Can I do the same chore multiple times in one day?

No. Each chore can only be claimed once per day per person. This prevents gaming the system.

What happens if I reject a chore?

The chore is marked as rejected, no points are awarded, and the cooldown is NOT triggered. The kid can claim it again immediately after fixing the issue.

How do ties work for GOAT?

Both players are declared CO-GOATs! The system handles ties automatically and both get the achievement credit.

Can I change point values?

Yes! Use the Admin panel to adjust chore point values if something feels too high or too low.

What if someone forgets to claim a chore?

Parents can create a custom task with the appropriate point value and assign it retroactively.

How often does the app refresh?

Automatically every 15 minutes. You can also manually refresh your browser anytime.

What's the maximum I can earn in a week?

Theoretically unlimited! But realistically, if you did every available chore, you could earn 4,000-6,000 points per week depending on cooldowns.
